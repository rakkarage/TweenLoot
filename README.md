# 🌀 TweenLoot

![screenshot1](screenshot1.png) ![screenshot2](screenshot2.png)

Apply tween to loot window.
Choose from many (all) easing types for:

- Alpha
- Scale
- Position

## Usage

- `/tl` (or `/tweenloot`) - Open TweenLoot options
- `/testnew` - Direct Tween Test
- `/testold` - Direct Normal Test
